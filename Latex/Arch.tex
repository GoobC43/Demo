\section*{1. System Objective}

Build a SaaS decision-support platform that:

\begin{enumerate}
    \item Continuously monitors maritime disruption signals
    \item Maps exposure across inbound SKUs
    \item Simulates mitigation strategies
    \item Optimizes trade-offs under company-specific constraints
    \item Generates explainable mitigation recommendations
\end{enumerate}

System must support:
\begin{itemize}
    \item Real-time disruption trigger detection
    \item Deterministic + probabilistic modeling
    \item Human-in-the-loop approval workflow
\end{itemize}

\subsection*{3.2 Exposure Mapping Engine}

\textbf{Purpose:} Map disruption to operational impact.

\textbf{Inputs:}
\begin{itemize}
    \item Shipment table
    \item SKU table
    \item Inventory levels
\end{itemize}

\textbf{Key Calculations:}

Inventory runway per SKU:

\[
Runway_i = \frac{Inventory_i}{DailyDemand_i}
\]

Delay impact:

\[
AdjustedArrival_i = OriginalArrival_i + ExpectedDelay
\]

Stockout condition:

\[
AdjustedArrival_i > Runway_i
\]

\textbf{Outputs:}
\begin{itemize}
    \item At-risk SKUs
    \item Containers impacted
    \item Revenue-at-risk estimate
\end{itemize}

\section*{4. Optimization Engine}

\subsection*{4.1 Objective Function}

For strategy $s$:

\[
\max_{s} \left( R_s - C_s - P_s \right)
\]

Where:

\begin{itemize}
    \item $R_s$ = Revenue preserved
    \item $C_s$ = Mitigation cost
    \item $P_s$ = SLA penalty cost
\end{itemize}

\subsection*{4.2 Revenue Preserved}

\[
R_s = \sum_{i \in SKUs} (Demand_i \times Margin_i \times Availability_{i,s})
\]

\subsection*{4.3 Mitigation Cost}

\[
C_s = AirCost_s + RerouteCost_s + BufferHoldingCost_s
\]

\subsection*{4.4 SLA Penalty Model}

If service level falls below threshold:

\[
P_s = \lambda \times (SLA_{target} - SLA_{achieved})
\]

Where $\lambda$ is penalty weight derived from Risk DNA.

\section*{5. Risk DNA Parameterization}

Each company has structured risk parameters:

\begin{itemize}
    \item $\alpha$ = Risk tolerance coefficient
    \item $\beta$ = SLA weight
    \item $\gamma$ = Working capital constraint
    \item $\delta$ = Customer churn sensitivity
\end{itemize}

Modified objective:

\[
\max_{s} \left( \alpha R_s - \beta C_s - \delta P_s \right)
\]

Subject to:

\[
WorkingCapital_s \leq \gamma
\]

\section*{6. Strategy Simulation}

Strategies modeled:

\begin{itemize}
    \item $S_1$ = No action
    \item $S_2$ = Full air freight
    \item $S_3$ = Port reroute
    \item $S_4$ = Split strategy
\end{itemize}

Monte Carlo simulation (optional for advanced model):

\[
Delay \sim \mathcal{N}(\mu, \sigma^2)
\]

Expected value over scenarios:

\[
E[Outcome_s] = \sum p_k \cdot Outcome_{s,k}
\]
